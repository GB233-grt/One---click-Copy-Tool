# 一键复制工具 - 标签页功能使用说明

## 一、功能概述

本应用程序采用标签页（Tab）形式展示多个文件内容，用户可以方便地在不同文件之间切换，实现高效的内容查看和复制操作。

## 三、标签页基本操作

### 3.1 添加文件到标签页

**操作步骤：**

1. 点击工具栏中的「添加文件」按钮
2. 在弹出的文件选择对话框中，按住 `Ctrl` 键（Windows/Linux）或 `Command` 键（macOS）选择多个文件
3. 点击「确定」按钮，选中的文件将以标签页形式打开

**或使用文件夹批量添加：**

1. 点击工具栏中的「添加文件夹」按钮
2. 在弹出的对话框中选择目标文件夹
3. 该文件夹中的所有文件将自动以标签页形式打开

### 3.2 切换标签页

**方法一：点击标签**
```
鼠标左键直接点击目标标签页的标签名称即可切换
```

**方法二：快捷键（需要配置）**
```
Ctrl+Tab：切换到下一个标签页
Ctrl+Shift+Tab：切换到上一个标签页
```

### 3.3 关闭标签页

**关闭当前标签页：**
1. 确保目标标签页处于选中状态
2. 点击工具栏中的「关闭当前」按钮

**关闭所有标签页：**
1. 点击工具栏中的「关闭所有」按钮
2. 所有文件标签页将被关闭，显示欢迎界面

## 四、标签页内容操作

### 4.1 查看文件内容

每个标签页显示对应文件的内容，包含：
- **行号**：显示在每行左侧，方便定位
- **内容**：显示在中间区域，使用等宽字体
- **复制按钮**：每行右侧的「Copy」按钮

### 4.2 单行复制

**操作步骤：**
1. 切换到目标文件的标签页
2. 找到需要复制的行
3. 点击该行右侧的「Copy」按钮
4. 状态栏显示「已复制 文件名 第X行」表示复制成功

### 4.3 多行/多文件复制

**复制所有文件内容：**
- 点击「复制所有文件」按钮
- 所有文件的内容将依次复制到剪贴板

**合并复制：**
- 点击「合并复制」按钮
- 在弹出的对话框中选择：
  - **是**：按文件顺序合并所有行，添加文件分隔符
  - **否**：每个文件内容单独分隔，用空行隔开

## 六、注意事项

### 6.1 标签页数量限制
- 建议同时打开的文件不超过10个，以保证界面响应速度
- 单个文件大小限制为10MB，超过限制的文件将无法加载

### 6.2 编码兼容性
- 仅支持 UTF-8 编码的文本文件
- 其他编码格式的文件可能无法正确显示，会提示「编码不支持」

### 6.3 权限问题
- 需要确保对目标文件和文件夹有读取权限
- 无权限访问的文件会提示「权限不足」

### 6.4 内存使用
- 打开大量文件时会占用较多内存
- 建议及时关闭不再需要的标签页

## 七、故障排除

### 标签页不显示
1. 检查是否已正确添加文件
2. 确认应用程序窗口大小足够显示标签页
3. 尝试调整窗口宽度

### 切换标签页无响应
1. 等待文件加载完成（状态栏显示「正在加载文件...」）
2. 检查是否有错误提示框未关闭

### 复制按钮失效
1. 确保该行内容存在
2. 检查系统剪贴板是否被其他程序占用
